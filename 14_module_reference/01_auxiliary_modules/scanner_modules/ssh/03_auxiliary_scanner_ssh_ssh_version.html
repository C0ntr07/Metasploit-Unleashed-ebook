auxiliary/scanner/ssh/ssh_version

From Metasploit Unleashed - Mastering The Framework

As far as protocols go, SSH is very secure but this does not mean that it has always been that way. There have been some instances where vulnerabilities have been found in SSH so it is always wise to look for older versions that have not yet been patched. The ssh_version is a very simple module that will scan a range of addresses and fingerprint the SSH version running on the remote host. 

msf > use auxiliary/scanner/ssh/ssh_version
msf auxiliary(ssh_version) > show options

Module options:

   Name     Current Setting  Required  Description
   ----     ---------------  --------  -----------
   RHOSTS                    yes       The target address range or CIDR identifier
   RPORT    22               yes       The target port
   THREADS  1                yes       The number of concurrent threads

msf auxiliary(ssh_version) > set RHOSTS 192.168.1.0/24
RHOSTS => 192.168.1.0/24
msf auxiliary(ssh_version) > set THREADS 255
THREADS => 255
msf auxiliary(ssh_version) > run
